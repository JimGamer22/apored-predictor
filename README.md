# 📈 ApoRed Predictor
Dieses Repository enthält den Code für den **ApoRed Predictor**, ein Tool zur Analyse und Vorhersage von Daten rund um ApoRed.
## 🚀 Live DemoDie Anwendung ist direkt im Browser verfügbar:
👉 **[ApoRed Predictor App](https://apored-predictor.streamlit.app/)**
## 📊 DatenquelleAlle zugrunde liegenden Daten werden in diesem Google Sheet gepflegt:
👉 **[Daten-Tabelle (Google Sheets)](https://docs.google.com/spreadsheets/d/1tPIJrC23fRyA7AFkqStnuq8r4WT_sB6ai91z6kVTpC0/edit?gid=0#gid=0)**
## 🛠️ Features- **Streamlit Integration:** Einfache und intuitive Weboberfläche.- **Datenvisualisierung:** Grafische Aufbereitung der aktuellen Statistiken.- **Vorhersagen:** Algorithmen zur Analyse zukünftiger Trends.
## 💻 Lokale Installation
Wenn du das Projekt lokal ausführen möchtest, folge diesen Schritten:
1. **Repository klonen:**
   ```bash
   git clone https://github.com
   cd apored-predictor


   1. Abhängigkeiten installieren:
   
   pip install -r requirements.txt
   
   2. App starten:
   
   streamlit run main.py
